Willkommen bei Smeagol, das Wiki zur Dokumentation in einem Repository.

Untenstehend siehst Du eine Liste aller Git Repositories auf Deinem Repository-Server auf die Du entsprechende Zugriffsrechte hast.
Dokumentation für ein Repository zu schreiben ist ganz einfach. Dafür muss eine .smeagol.yml Datei in das Root-Verzeichnis des Repository abgelegt werden. Anschließend kann das Wiki in diesem Repository direkt genutzt werden. Alle Änderungen in einem Wiki werden nach dem Speichern von Smeagol an das Repository gepusht. Damit bleibt die Dokumentation stets aktuell und genau an dem Ort zu dem sie gehört - direkt neben dem Code!
